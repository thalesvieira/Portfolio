# 📊 Data Analyst & RPA Portfolio — Thales Von Arnt

Este repositório contém o código-fonte do meu portfólio profissional, desenvolvido em HTML/CSS com visualização de dados dinâmica utilizando **Chart.js**. O objetivo é demonstrar minha jornada técnica, proficiência em ferramentas de BI e impacto gerado através de automações.

📍 **Acesse o portfólio online:** [https://thalesvieira.github.io/curriculo/]

---

## 🚀 Tecnologias em Destaque

O portfólio apresenta uma visão analítica das seguintes competências:

* **Business Intelligence:** Dashboards estratégicos com Power BI (DAX e Power Query).
* **Automação (RPA):** Desenvolvimento de bots com Python (Pandas, Selenium) para otimização de processos.
* **Engenharia de Dados:** Estruturação de pipelines ETL e administração de bancos de dados SQL Server/MySQL.
* **Desenvolvimento:** Experiência em C#/.NET, Python e integrações via APIs REST.

## 📈 Visualizações de Dados Inclusas

Diferente de um currículo estático, este projeto utiliza gráficos interativos para demonstrar:
1.  **Radar de Proficiência:** Nível técnico em stacks de dados e desenvolvimento.
2.  **Distribuição de Atuação:** Foco de carreira entre BI, RPA e Dev.
3.  **Evolução Tecnológica:** Histórico de dedicação às tecnologias ao longo dos últimos 6 anos.

## 🛠️ Como este projeto foi construído

* **Linguagem:** HTML5 e CSS3 (Design Responsivo e Dark Mode).
* **Gráficos:** [Chart.js](https://www.chartjs.org/) para visualização de dados dinâmica.
* **Animações:** Intersection Observer API para efeitos de scroll e contadores dinâmicos.
* **Hospedagem:** GitHub Pages.
